## 一、设计背景

### 1.1 流派定位

超能属性在循迹之境的8种属性体系中，定位为**奥秘流/心理博弈**流派。

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **超能** | 奥秘流 / 心理博弈 | 心灵异能的迷雾，让对手在猜谜中犯错 | 心理博弈 | ★★★★★ |

### 1.2 设计理念（修订版）

> *一句话定位：心灵异能的迷雾，让对手在猜谜中犯错*

**核心机制**（修订/新增）：
- **预言标记**：使用超能技能时获得标记，标记层数增强「存储力量」等技能威力
- **精神场地**：超能属性独有的环境系统，己方超能技能威力+30%
- **延迟触发**：效果在N回合后触发或满足条件时触发
- **信息操控**：干扰敌方对意图图标的解读

**节奏特征**：前期布阵（积累预言标记） → 中期发酵 → 后期爆发

### 1.3 新增核心概念

#### 1.3.1 精神场地

精神场地是超能系独有的战场状态，效果包括：
- 己方超能系技能伤害 **+30%**
- 保护己方免受优先度攻击

#### 1.3.2 预言标记

核心机制：使用超能技能时获得「预言标记」。

| 预言层数 | 存储力量威力加成 |
|----------|------------------|
| 1层 | +20威力 |
| 2层 | +40威力 |
| 3层 | +60威力 |
| N层 | +20×N威力 |

## 二、完整技能设计（13种）

### 2.1 攻击倾向技能（5种）

#### 【攻击倾向1】迷心刺

| 属性 | 值 |
|------|-----|
| **技能ID** | `mind_pierce` |
| **技能名称** | 迷心刺 |
| **能量消耗** | 2能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 60 |
| **伤害类型** | 物理伤害（超能属性） |

**效果描述**：攻击单体目标，造成60威力物理伤害，获得1层「预言标记」，附加「心灵创伤」（持续2回合）

**核心机制**：
- 造成伤害后，自身获得1层「预言标记」
- 附加「心灵创伤」：目标攻击命中率50%击中自己

**设计理由**：
- 60威力对应2能量，标准输出技能
- 获得预言标记配合「存储力量」形成连招
- 心灵创伤是混乱的变体，增加心理压力

#### 【攻击倾向2】精神冲击

| 属性 | 值 |
|------|-----|
| **技能ID** | `psychic_hit` |
| **技能名称** | 精神冲击 |
| **能量消耗** | 3能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 80 |
| **伤害类型** | 特殊伤害（超能属性） |

**效果描述**：攻击单体目标，造成80威力混合伤害（特攻计算/物防减免），无视当前护盾，可破坏屏障类效果

**核心机制**：
- 伤害计算使用特攻，但防御计算使用物防
- 无视目标当前护盾值，直接造成伤害
- 可破坏屏障类效果（如光墙、反射等）

**设计理由**：
- 高威力（80）对应高消耗（3能量）
- 无视护盾是针对防御流的反制设计
- 混合伤害机制增加伤害输出的稳定性

## 五、战斗流程配合

### 5.1 推荐起手策略

| 回合 | 建议技能 | 理由 |
|------|----------|------|
| 第1回合 | 精神场地（8能量） | 建立环境基础 |

| 第5回合 | 预知未来/虚空预言 | 收割阶段 |

### 5.2 预言标记连招

```
标准累积连招：
第1回合：精神场地（8能量）- 超能威力+30%
第2回合：迷心刺（2能量）- 60伤害 + 预言标记1层
```

## 七、核心机制详解

### 7.1 精神场地机制

**召唤方式**：「精神场地」

**持续时间**：5回合

**叠加效果**：不可叠加，后使用者刷新时间

**环境效果**：
1. 己方超能系技能伤害 +30%
2. 保护己方免受优先度攻击

### 7.2 预言标记机制

**触发条件**：使用超能技能后获得

**效果**：增强「存储力量」技能威力（每层+20）

**清空条件**：
- 战斗结束后清空
- 离开战斗后清空

**可叠加**：上限10层
