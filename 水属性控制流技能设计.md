## 一、设计背景

### 1.1 流派定位

水属性在循迹之境的8种属性体系中，定位为**控制流**流派。

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **水** | 控制流 | 以静制动的深潭宰相，用韧性消磨一切攻势 | 持久消耗 | ★★★★☆ |

### 1.2 设计理念

> *一句话定位：以静制动的深潭宰相，用韧性消磨一切攻势*

**核心机制**：
- **治疗续航**：持续恢复HP，维持长期作战能力
- **护盾保护**：吸收伤害的护盾机制
- **软解控制**：减速、DOT、溺水等软控制效果
- **持久消耗**：用积累的HP差碾压对手

**节奏特征**：中期建立资源优势 → 后期用积累的HP差碾压

### 1.3 流派克制关系

| 攻击流派 | 克制 | 被克制 | 设计理由 |
|----------|------|--------|----------|
| **控制流（水）** | 爆发流、连击流 | 光环流 | 控制技能打断连击节奏，被持久消耗拖垮 |

### 1.4 属性克制关系

| 水属性 | 效果 |
|--------|------|
| **克制** | 火（灭火降温）、岩石（侵蚀冲刷） |
| **被克制** | 草（吸水生长）、电（导体导电） |

## 二、完整技能设计（10种）

### 2.1 攻击倾向技能（4种）

#### 【攻击倾向1】水流冲击

| 属性 | 值 |
|------|-----|
| **技能ID** | `water_jet` |
| **技能名称** | 水流冲击 |
| **能量消耗** | 2能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 60 |
| **伤害类型** | 特殊伤害（水属性） |

**效果描述**：攻击单体目标，造成60威力水属性伤害，**目标获得**「浸透」

**附加效果**：
- 目标获得「浸透」状态（特防-1级/层，最多6层）
- 浸透持续2回合，每次叠加刷新持续时间

**设计理由**：
- 中等威力（60）对应中等消耗（2能量）
- 浸透debuff让水属性有削弱目标的控制机制
- 最高6层提供-6级特防的强力削弱
- 为后续技能铺垫优势局面

**代码实现**：
```typescript
export const WATER_JET: Skill = (() => {
  const definition: SkillDefinition = {
    id: 'water_jet',
    name: '水流冲击',
    description: '攻击单体目标，造成60威力水属性伤害，目标获得「浸透」（特防-1级/层，最多6层，持续2回合）',
    type: 'action',
    energyCost: EnergyCost.MEDIUM,
    target: SkillTarget.SINGLE,
    tendency: SkillTendency.ATTACK,
    effects: [{
      damage: {
        basePower: 60,
        damageType: DamageType.SPECIAL,
        element: ElementType.WATER
      },
      applyDebuff: {
        debuffType: 'water_soak',
        duration: 2,
        stacks: 1,
        maxStacks: 6
      }
    }],
    category: '水属性控制流·攻击',
    tags: ['水', '控制流', '攻击', '削弱', '浸透']
  };
  return new Skill(definition);
})();
```

### 2.2 防御倾向技能（2种）

#### 【防御倾向1】水之守护

| 属性 | 值 |
|------|-----|
| **技能ID** | `aqua_shield` |
| **技能名称** | 水之守护 |
| **能量消耗** | 2能量 |
| **目标类型** | 自身 |
| **持续时间** | 1回合 |

**效果描述**：受到伤害降低70%（持续1回合），如果受到伤害，**攻击者**获得1层浸透

**附加效果**：
- 获得「减伤」状态：受到伤害降低70%（持续1回合）
- 反击触发：每受到1次伤害，攻击者获得1层浸透（特防-1级/层）

**设计理由**：
- 短期高额减伤（70%）提供紧急生存能力
- 受击时使攻击者获得浸透是反击式削弱
- 可以配合水流冲击快速叠加目标浸透层数
- 防御技能转型为进攻增益手段

**代码实现**：
```typescript

        stacks: 1,
        maxStacks: 6
      }
    },
    category: '水属性控制流·防御',
    tags: ['水', '控制流', '防御', '减伤', '浸透']
```
