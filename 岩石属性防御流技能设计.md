## 一、设计背景

### 1.1 流派定位

岩石属性在循迹之境的8种属性体系中，定位为**防御流**。

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **岩石** | 防御流 | 坚如磐石的铜墙铁壁，让敌人绝望的叹息 | 以守为攻 | ★★☆☆☆ |

### 1.2 设计理念

> *一句话定位：坚如磐石的铜墙铁壁，让敌人绝望的叹息*

**核心机制**：
- **高防御**：通过技能直接提升防御属性
- **固定减伤**：护盾和减伤机制降低受到的伤害
- **护甲积累**：持久战中防御优势逐渐累积
- **反击反弹**：受到攻击时概率反弹伤害

**节奏特征**：全程防御积累 → 反打收割

### 1.3 流派克制关系

| 攻击流派 | 克制 | 被克制 | 设计理由 |
|----------|------|--------|----------|
| **防御流（岩石）** | 连击流 | 控制流、光环流 | 被软控制消耗致死 |

**属性克制**：
- 克制：火（耐高温）、冰（耐寒霜）
- 被克制：水（侵蚀溶解）、草（根穿岩缝）

### 2.1 攻击倾向技能（4种）

#### 【攻击倾向1】尖石攻击

| 属性 | 值 |
|------|-----|
| **技能ID** | `rock_spike` |
| **技能名称** | 尖石攻击 |
| **能量消耗** | 2能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 75 |
| **伤害类型** | 物理伤害（岩石属性） |

**效果描述**：攻击单体目标，造成75威力岩石伤害，30%概率使目标防御-1级

**附加效果**：
- 30%概率使目标防御-1级（持续3回合）

**设计理由**：
- 中威力（75）对应中等消耗（2能量）
- 防御下降是软控制，配合防御流反击
- 物理伤害契合防御流的攻击风格

**代码实现**：
```typescript
export const ROCK_SPIKE: Skill = (() => {
  const definition: SkillDefinition = {
    id: 'rock_spike',
    name: '尖石攻击',
    description: '攻击单体目标，造成75威力岩石伤害，30%概率使目标防御-1级',
    type: 'action',
    energyCost: EnergyCost.MEDIUM,
    target: SkillTarget.SINGLE,
    tendency: SkillTendency.ATTACK,
    effects: [{
      damage: {
        basePower: 75,
        damageType: DamageType.PHYSICAL,
        element: ElementType.ROCK
      }
    }],
    category: '岩石防御流·攻击',
    tags: ['岩石', '防御流', '攻击', '防御下降']
  };
  return new Skill(definition);
})();
```

| `STONE_REFLECT` | 岩盾反弹 | 反弹15%伤害给攻击者 | 岩盾 |

## 五、战斗流程配合

### 5.1 推荐起手策略

| 回合 | 建议技能 | 理由 |
|------|----------|------|
| 第1回合 | 岩盾/铁壁 | 建立防御基础 |
| 第2回合 | 沙尘掩体/磐石之躯 | 增强生存+清除负面 |
| 第3回合 | 尖石攻击/反击姿态 | 积累软控制+反击 |
| 第4回合 | 磐石崩落蓄力 | 准备爆发 |
| 第5回合 | 磐石崩落/落石冲击 | 收割 |

### 5.2 流派协同

| 协同组合 | 效果 | 应用场景 |
|----------|------|----------|
| 岩盾 + 反击姿态 | 双重反弹机制 | 最大化反伤效果 |
| 铁壁 + 磐石之躯 | 防御+减伤叠加 | 面对高攻敌人 |
| 山岳守护 + 铁壁 | 群体护盾+单体强化 | 团队防御 |
| 沙尘掩体 + 磐石崩落 | 闪避+蓄力 | 保护蓄力期 |

### 5.3 对抗不同流派

| 对抗流派 | 推荐技能 | 策略 |
|----------|----------|------|
| **爆发流（火）** | 岩盾、反击姿态、沙尘掩体 | 利用护盾和闪避消耗爆发 |
| **连击流（电）** | 铁壁、山岳守护 | 硬抗高频攻击 |
| **控制流（水/冰）** | 磐石之躯、沙尘掩体 | 净化状态+减伤 |
| **光环流（草）** | 磐石崩落、地震 | 速战速决，避免拖入后期 |
