## 一、设计背景

### 1.1 流派定位

冰属性在循迹之境的8种属性体系中，定位为**冻结控制+破冰爆发**流派。

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **冰** | 冻结破冰流 | 以绝对零度冻结敌人，以破冰斩毁灭目标 | 冻结→爆发 | ★★★☆☆ |

### 1.2 设计理念

> *一句话定位：冻结即审判，破冰即毁灭*

**核心机制**：
- **冻结控制**：利用冰冻效果完全阻止敌人行动
- **破冰爆发**：对冻结目标造成毁灭性伤害
- **碎冰连击**：冻结破碎时触发额外追击

**节奏特征**：冻结铺底 → 冻结累积 → 破冰收割

### 1.3 流派克制关系

| 攻击流派 | 克制 | 被克制 | 设计理由 |
|----------|------|--------|----------|
| **冻结破冰流（冰）** | 草、龙 | 火、岩石 | 冻死植物、限制龙族行动 |

### 1.4 属性克制关系

| 攻击方\防御方 | 火 | 水 | 草 | 电 | 冰 | 岩 | 龙 | 超能 |
|--------------|-----|-----|-----|-----|-----|-----|-----|------|
| **冰** | ×0.5 | ×1 | ×2 | ×1 | ×0.5 | ×1 | ×2 | ×1 |

## 二、核心机制说明

### 2.1 冻结状态

| 属性 | 值 |
|------|-----|
| **效果** | 完全无法行动 |
| **持续时间** | 2回合 |
| **解冻概率** | 每回合20%自动解冻 |
| **受伤解除** | 受到伤害时必定解除冻结 |

### 2.2 破冰倍率

对已冻结目标造成的首次攻击享受伤害加成：

| 条件 | 加成 | 说明 |
|------|------|------|
| 目标冻结中 | **+80%** | 破冰伤害爆炸 |
| 刚解冻 | +30% | 残寒追击 |
| 碎冰追击 | 触发额外追击 | 冻结破碎时 |

### 2.3 冻结等级

| 等级 | 解冻抗性 | 说明 |
|------|----------|------|
| 普通冻结 | 20%自解 | 标准冻结 |
| 深冻 | 10%自解 | 更难解除 |
| 绝对冻结 | 0%自解 | 必须受伤解除 |

## 三、完整技能设计（10种）

### 3.1 攻击倾向技能（4种）

#### 【攻击倾向1】冰晶射击

| 属性 | 值 |
|------|-----|
| **技能ID** | `ice_shot` |
| **技能名称** | 冰晶射击 |
| **能量消耗** | 1能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 40 |
| **特效** | 先手+冻结铺垫 |

**效果描述**：如闪电般快速的射击，造成40威力冰属性伤害，**15%概率冻结目标**，先手攻击

**设计理由**：
- 1能量先手技能，抢占先机
- 冻结铺垫，为后续技能创造机会
- 参考宝可梦冰锥(Ice Shard)

---

#### 【攻击倾向2】霜冻之息

| 属性 | 值 |
|------|-----|
| **技能ID** | `frost_breath` |
| **技能名称** | 霜冻之息 |
| **能量消耗** | 2能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 60 |
| **冻结概率** | 25% |

**效果描述**：喷吐刺骨寒气，造成60威力冰属性伤害，**25%概率使目标冻结**

**设计理由**：
- 稳定冻结技能
- 2能量中等消耗
- 冻结控制的核心铺垫

---

#### 【攻击倾向3】破冰斩

| 属性 | 值 |
|------|-----|
| **技能ID** | `ice_shatter` |
| **技能名称** | 破冰斩 |
| **能量消耗** | 3能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 70（基础）/ **126**（破冰） |
| **特效** | 破冰倍率+碎冰追击 |

**效果描述**：对目标发动毁灭性斩击。造成70威力冰属性伤害。**若目标已冻结，伤害提升至126（+80%），并触发碎冰追击（追加40威力）**

**特殊机制**：
- 破冰倍率：冻结目标+80%伤害
- 碎冰追击：破冰成功后追加40威力追击
- 总伤害：70基础 → 126破冰 → 166碎冰

**设计理由**：
- 流派核心爆发技能
- 破冰+碎冰追击连段
- 高风险高回报（需要先冻结目标）

---

#### 【攻击倾向4】绝对零度

| 属性 | 值 |
|------|-----|
| **技能ID** | `absolute_zero` |
| **技能名称** | 绝对零度 |
| **能量消耗** | 5能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 150 |
| **特效** | 蓄力+必定冻结+深冻 |

**效果描述**：蓄力1回合后发动，造成150威力冰属性伤害，**100%概率使目标进入「深冻」状态**

**蓄力机制**：
- 蓄力期间若被攻击则技能取消
- 蓄力成功后必定冻结目标

**深冻效果**：
- 持续3回合
- 每回合仅10%概率自动解冻
- 受伤时解除

**设计理由**：
- 终极单体爆发+控制技能
- 深冻让敌人长时间无法行动
- 高风险高回报的蓄力机制

## 八、设计总结

### 8.1 设计亮点

1. **冻结+破冰核心闭环**：从冻结铺垫到破冰爆发的完整战斗循环
2. **破冰倍率极高**：冻结目标+80%伤害，激励玩家主动创造冻结机会
3. **碎冰追击机制**：破冰成功后追加伤害，连招感强
4. **三级冻结体系**：普通/深冻/绝对冻结，提供差异化控制深度
