## 一、设计背景

### 1.1 流派定位

火属性在循迹之境的8种属性体系中，定位为**爆发流/OTK流**。

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **火** | 爆发流 / OTK流 | 烈焰焚城的暴力美学，一击必杀的极致追求 | 速战速决 | ★★★☆☆ |

### 1.2 设计理念

> *一句话定位：烈焰焚城的暴力美学，一击必杀的极致追求*

**核心机制**：
- **高伤害单体爆发**：牺牲持续性换取瞬间伤害
- **蓄力→爆发**：蓄力后伤害大幅提升，但蓄力期间脆弱
- **灼烧DOT**：通过灼烧状态提供持续伤害压力

**节奏特征**：前期蓄力（脆弱期）→ 中期压制 → 一回合秒杀

### 1.3 流派克制关系

| 攻击流派 | 克制 | 被克制 | 设计理由 |
|----------|------|--------|----------|
| **爆发流（火）** | 防御流 | 控制流 | 防御流拖慢节奏，爆发流无法蓄力 |

**属性克制**：
- 克制：草（干枯助燃）、冰（融化成水）
- 被克制：水（灭火）、岩（隔绝氧源）

## 二、完整技能设计（10种）

### 2.1 攻击倾向技能（4种）

#### 【攻击倾向1】点燃

| 属性 | 值 |
|------|-----|
| **技能ID** | `ember` |
| **技能名称** | 点燃 |
| **能量消耗** | 1能量 |
| **目标类型** | 单体敌人 |
| **伤害类型** | 无直接伤害 |

**效果描述**：对目标施加灼烧（5层）

**附加效果**：
- 使目标陷入「灼烧」状态（持续3回合，5层）
- 灼烧：每层回合结束时造成2%最大HP伤害，层数减半

**设计理由**：
- 纯DOT技能，提供持续伤害压力
- 5层灼烧配合层数减半机制：5→2→1→1→0（4回合DOT）

**代码实现**：
```typescript
export const EMBER: Skill = (() => {
  const definition: SkillDefinition = {
    id: 'ember',
    name: '点燃',
    description: '对目标施加灼烧（5层）【每层2%最大HP伤害，层数减半】',
    type: 'action',
    energyCost: EnergyCost.LOW,
    target: SkillTarget.SINGLE,
    tendency: SkillTendency.ATTACK,
    effects: [{
      applyDebuff: {
        debuffType: 'burn',
        duration: 3,
        stacks: 5,
        successRate: 1.0
      }
    }],
    category: '火属性爆发流·攻击',
    tags: ['火', '爆发流', '攻击', '灼烧']
  };
  return new Skill(definition);
})();
```

#### 【攻击倾向2】烈焰拳

| 属性 | 值 |
|------|-----|
| **技能ID** | `flame_punch` |
| **技能名称** | 烈焰拳 |
| **能量消耗** | 2能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 25×3（共75威力） |
| **伤害类型** | 物理伤害（火属性） |
| **特殊标签** | `hits`（连击3次） |

**效果描述**：连续3次火属性攻击（共75威力），每次命中50%概率灼烧（2层）

**附加效果**：
- 连续3次攻击，每次50%概率使目标灼烧（2层）
- 灼烧：每层回合结束时造成2%最大HP伤害，层数减半

**设计理由**：
- 多段攻击与电属性的雷霆连击形成差异化（电强调追加攻击，火强调灼烧压力）
- 每次独立判定灼烧，累积概率更高
- 物理伤害类型保持一致
- 50%×3次 ≈ 87.5%至少命中一次灼烧

      }
    }],
    category: '火属性爆发流·攻击',

| `FLAME_BODY` | 烈火护体 | 受到伤害-70%，下次火攻必定暴击 | 烈火护体 |
| `FLAME_CHARGE` | 蓄焰 | 下次火属性攻击威力+50% | 蓄焰 |
| `BLAZE_WILL` | 炎之意志 | 攻击+1级，速度+1级，火属性伤害+15% | 炎之意志 |

### 4.2 Debuff类型（3种）

| Debuff类型 | 名称 | 效果描述 | 关联技能 |
|------------|------|----------|----------|
| `BURN` | 灼烧 | 每层回合结束时造成2%最大HP伤害，层数减半，持续3回合 | 点燃（5层）、烈焰拳（2层）、爆炸烈焰（4层） |

| `COMBUSTION_MARK` | 燃尽印记 | 3回合后扣除30%当前HP | 燃尽 |

## 五、战斗流程配合

### 5.1 推荐起手策略

| 回合 | 建议技能 | 理由 |
|------|----------|------|
| 第1回合 | 烈焰壁垒/火盾 | 建立防御基础，保护脆弱期 |
| 第2回合 | 蓄焰/烈火护体 | 为爆发做准备+减伤保护 |
| 第3回合 | 烈焰拳/点燃 | 积累伤害和灼烧DOT |
| 第4回合 | 大字爆炎/燃尽 | 压低血线 |
| 第5回合 | 爆炸烈焰/炎之意志 | 收割或群体强化 |

### 5.2 爆发流黄金组合

| 协同组合 | 效果 | 应用场景 |
|----------|------|----------|
| 蓄焰 + 爆炸烈焰 | 蓄焰+50% → 爆炸烈焰150×1.5=225威力 | 一击必杀 |
| 蓄焰 + 大字爆炎 | 蓄焰+50% → 大字爆炎120×1.5=180威力 | 高伤害收割 |
| 烈焰壁垒 + 蓄力 | 抗性+30%保护蓄力期 | 应对草/冰敌人 |
| 炎之意志 + 大字爆炎 | 全队火伤+15% → 大字爆炎120×1.15=138威力 | 群体强化收割 |
| 燃尽 + 爆炸烈焰 | 燃尽30%HP → 爆炸烈焰收割 | 百分比斩杀 |

### 5.3 OTK流完整连招

```
回合1：烈焰壁垒（建立防御）
      ↓
回合2：蓄焰（为核心单位施加增伤）
      ↓
回合3：等待敌人行动（蓄焰单位可能被攻击，触发火盾反伤）
      ↓
回合4：大字爆炎/燃尽（压低血线）
      ↓
回合5：爆炸烈焰（蓄焰加成后225威力，一击必杀）
```

## 七、能量消耗设计说明

### 7.1 能量-威力对照表

| 能量消耗 | 威力范围 | 设计技能 | 说明 |
|----------|----------|----------|------|
| **0能量** | ≤40 | — | 免费技能（未使用） |
| **1能量** | DOT | 点燃 | 纯DOT技能 |
| **2能量** | 60-80 | 烈焰拳、蓄焰 | 中等威力/辅助 |
| **3能量** | 80-90 | 烈焰壁垒、烈火护体 | 高消耗防御 |
| **4能量** | 100-130 | 大字爆炎、燃尽 | 超高威力/特殊机制 |
| **5能量** | 130-160 | 爆炸烈焰、炎之意志 | 终极技能 |

### 7.2 爆发流能量设计原则

- **高频使用**：低能量技能（1-2能量）保证每回合都有事可做
- **爆发准备**：辅助技能（2-3能量）为高威力技能铺垫
- **一击必杀**：终极技能（5能量）提供极限伤害

## 九、平衡性调整指南

### 9.1 如果爆发流过强

- 降低大字爆炎威力（120→110）
- 降低蓄焰增伤幅度（50%→40%）
- 增加爆炸烈焰蓄力回合数（1→2）

### 9.2 如果爆发流过弱

- 提高大字爆炎威力（120→130）
- 提高蓄焰增伤幅度（50%→60%）
- 降低爆炸烈焰能量消耗（5→4）

### 9.3 防御技能平衡

- 火盾55%减伤+灼烧扩散可根据实际战斗数据调整
- 烈火护体70%减伤可根据实际战斗数据调整
