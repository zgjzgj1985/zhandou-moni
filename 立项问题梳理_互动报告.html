一、观察与思考
品类背景

洛克王国世界（2026年3月公测）是中国大陆市场回合制宠物收集战斗品类表现最好的产品，上线首月DAU均值1300万（腾讯2026年Q1财报披露），前10天iOS收入约1.03亿元（第三方数据平台估算）。

PVE内容观察
  • 游戏基础PVE关卡难度偏低——属性克制+速通组合（如"迪莫，罗隐"）可快速通关，缺乏持续挑战压力
  • 官方后续补了高难PVE（如"领地试炼"、"星光对决"），但整体仍未形成支撑长线的PVE生态闭环（领地试炼难度过高，普通人没有抄作业的地方，星光对决只是一个相对简单的周长，没有技巧只有数值）
  • PVE内容设计空洞：核心循环只有清体力+抓异色，S1、S2赛季都存在这个问题；S1靠新鲜感+大量主线+图鉴任务掩盖了问题，S1到S2内容重复单调，战斗无聊；缺乏目标感——无论是内在动机还是外在动机都没有分层设计（短期/中期/长期）；唯一的外在动机是极低概率的野生炫彩，由于概率太低对收集党/外观党来说动机也很弱
  • 抓异色是游戏最重要的玩法，但设计粗糙：
    • S1赛季"污染"机制和S2赛季"惊喜盒子"都存在严重问题
    • 以S2盒子为例：玩家通过互动可提前确定盒子内是否有异色，打开盒子进入战斗变成0收益环节——违反"悬念留到最后"的设计原则
    • 长达1000+只的抓捕过程中没有任何收益分层，只有获得异色时才会获得1次强烈的正反馈，其他时间就是纯粹的坐牢（重复劳动）
    • 优化方向：收益分层设计——每50只左右给L1收益、200只给L2收益、1000只给L3（异色），且消除悬念提前等问题
  • 洛克王国世界的社交是自发式的、有温度但无目的的社交——没有实质的多人玩法，只有玩家间的松散互动。可以引入实质的多人玩法（区域暴君等高强度PVE目标），让战斗策略空间得以扩展、挑战上限得以提升；同时也让外观党有了一个展示战果的舞台。
  • 生物生态做得非常好，已经是行业天花板。这是巨大工程量累积的成果。
  • 剧情偏子供向，部分玩家觉得无聊且节奏被拖慢；剧情冗长且不可跳过，中后期跳过按钮消失
  • 支线任务几乎都是跑图/点击流程，有明显赶工痕迹
PVP内容观察
  • PVP的深层问题在于机制设计：1V1轮换+猜拳+属性克制，理解门槛高；需要持续掌握属性克制关系和大量宠物的技能组；环境变化快，信息差持续存在；1V1无数值碾压、无随机性，失败几乎都源于决策，挫折感直接且强烈——PVP注定是非主流的硬核内容
  • PVP目前做得好的地方在于猜拳博弈成功的正反馈很足，靠智力战胜对手的体验感也很棒（虽然双方队伍构建极大影响胜负关系）。喜欢PVP的玩家可以连续几个小时进行对战；另外PVP战斗有一定观赏性，B站上有大量相关视频
  • 平衡性问题严重：强势精灵（如独角兽）一回合启动、两回合推队
相关思考
  • 官方重点放在了IP与生态构建，社交与PVP，PVE堆积了大量问题。从这些问题出发寻找适合的切入点。
  • 以赛季作为更新周期，持续提供有趣+有深度的PVE战斗内容。以有趣的战斗为载体，外在目标为驱动，用户分层+目标分层的方式设计内容。（这里的主要问题包含2个：战斗系统本身质量，赛季内容制作周期与设计质量）
  • 抓捕异色玩法的游戏体验完全可以做得更好。不依赖团队规模，只和设计框架，细节与执行落地有关。
  • 不考虑生态构建，开放世界（考虑英勇之地手游的箱庭式探索地图）。采取错位竞争而不是正面对决，找到自身在该品类的生态位。
  • 设计偏成年人（18-35）的剧情，叙事。
重点解决问题

持续提供有趣+有深度的PVE战斗内容：3Vn多对多战斗结构 + 意图可视化 + 芳香剂借用机制。战斗系统既是核心体验，也是差异化卖点。

  • 3Vn多对多战斗：相比宝可梦1V1，带来"集火谁、如何站位"的多目标决策维度——不是更多的宠，而是完全不同的博弈结构
  • 意图可视化：将敌人的行动意图透明化，让推理过程可见可交互，降低推理门槛的同时保留了推理本身的乐趣
  • 芳香剂借用机制：玩家可以接管敌人（尤其是BOSS）的行动权，借用其强力技能为我所用。在首领战中尤其有趣：BOSS拥有罕见的强力技能（无敌、复活等），通过芳香剂可以让这些技能作用于我方——这是正常对战里不可能获得的能力，同时也是"接管体验"趣味性的来源

B1群与循迹之境的"战斗即推理"定位存在天然矛盾。循迹之境的战斗系统（意图可视化、属性克制、气味干扰）本质上是提高认知门槛，对于纯B1群来说不是"有趣"而是"复杂"。典型的B1群用户想的是"我的小火猴好可爱"而不是"对面意图是强化还是输出"。

五、设计原则总结
设计原则不是功能清单，而是约束规则——规定"做什么、不做什么"，在具体决策时给出判断依据。
一
悬念留到最后
关键奖励或体验的揭晓时机必须在最后。在玩家确认结果之前，所有信息泄露都会消解行动的意义。
→ 收益节点可以提前告知（让玩家有目标），但结果揭晓不能提前（让过程有悬念）。
二
长线行为必须有多个收益节点
任何需要重复N次才能完成的行为，每隔N/10~N/5次就必须给出次级收益，让玩家在过程中有盼头。
→ 异色是L3终极大奖，但L1（小惊喜）和L2（大期待）必须穿插其中，让1000次捕获变成一段有起伏的旅程。
三
不讨好所有人
设计必须有明确取舍。试图满足所有人，结果是所有人都只能得到"还可以"而不是"很满意"。
→ 循迹之境的核心是战斗推理，主打中重度用户。B1群中的战斗兴趣者可转化，但不能为此稀释核心体验。
四
承认硬核内容的天花板
某些内容天然只适合小众。不要试图扩大受众，而是服务好核心受众，让他们愿意长期留存。
→ PVP可以存在，但资源投入要与受众规模匹配；区域暴君等高强度PVE目标同理。
五
外在动机与内在动机必须同时存在
每个内容层都要同时提供两种动机：外在动机（奖励/成就）告诉玩家"值不值得做"，内在动机（过程体验）告诉玩家"做起来好不好玩"。
→ 只有奖励没有体验=上班打卡；只有体验没有奖励=没有目标感。
六
错位竞争，不正面对决
找到自己的生态位，在竞品的痛点上建立优势，而不是在竞品的强项上追赶。
→ 循迹之境的竞争维度是"战斗深度"，不是"生态丰富度"。箱庭探索而非开放世界，战斗即推理而非1V1猜拳。
七
宠物是情感载体，不是数值工具
宠物不只是一个战斗力数值，要让玩家和宠物建立情感连接。数值可以成长，但情感连接才决定留存。
→ 外观系统（炫彩、异色）不只是审美满足，更是"我的宠物与众不同"的情感锚点。

快速决策清单

面对任何设计决策时，用这些问题快速判断：

问题 是 否
这个奖励的结果是提前揭晓还是留到最后？ ⚠️ 违反原则一 ✅ 通过
这个重复行为有多少个次级收益节点？ 太少→违反原则二 ✅ 通过
我们是为了讨好某个大用户群而妥协核心体验吗？ 是→违反原则三 ✅ 通过
这个硬核内容的受众规模是否与投入资源匹配？ 不匹配→违反原则四 ✅ 通过
这个内容有外在动机吗？有内在动机吗？ 缺任何一个→违反原则五 ✅ 通过
我们是在竞品的强项上竞争还是痛点上建立优势？ 强项竞争→违反原则六 ✅ 通过
这个设计让宠物更像数值工具还是更像情感载体？ 数值工具→违反原则七 ✅ 通过
六、还需要回答的问题

问题一：如何在前期留住玩家？（前15分钟）

核心设计思路

以追踪气味的剧情线作为前期叙事主轴，将"理解战斗机制"包装为"追踪猎物的本能"。

玩家不是"在学习游戏"，而是"在追踪一个特别的气味"——战斗操作是追踪的手段，捕获是追踪的结果。

1. 前15分钟的叙事锚点：追踪

整个前期内容以一个特别的气味作为叙事锚点驱动：

时间叙事节点游戏内容核心体验
0-2分钟嗅到异常气味并追踪玩家在野外追踪 → 发现气味源建立"气味=线索"的直觉
2-5分钟遭遇同样被气味吸引的生物遭遇第一场战斗（普通野兽）理解基础操作（攻击、技能）
5-8分钟发现气味来源追踪途中发现一枚特殊花纹的宠物蛋发现惊喜 + 情感锚点植入
8-11分钟偷取并逃跑引导玩家偷蛋 → 偷蛋成功悬念留到最后（蛋的价值揭晓在孵化时）
11-15分钟遭遇愤怒的巢穴主人巢穴主人返回 → 精英战斗意图可视化顿悟时刻

2. 以追踪气味为线索的内容链

进入游戏 → 玩家本能地追踪气味
↓ 【气味线索：地上有新鲜的痕迹】
遭遇同样被气味吸引的生物（战斗教学）
↓ 【气味线索：气味汇聚在巢穴方向】
发现气味来源——一枚特殊花纹的宠物蛋（惊喜植入）
↓ 【气味线索：趁主人不在拿走它】
偷取并逃跑（获得第一只宠物的伏笔）
↓ 【气味线索：愤怒的气息正在逼近】
遭遇愤怒的巢穴主人 → 精英战斗（需要理解意图）
↓
击败/击退巢穴主人 → 悬念钩子（"蛋里是什么？"）

3. 每个节点的"钩子"设计

节点玩家此刻的疑问设计给出的答案
追踪气味"这个气味是什么？"地上有可视化的气味痕迹，引导玩家跟随
遭遇被气味吸引的生物"有什么东西也在追踪这个气味？"触发第一场战斗教学
发现宠物蛋"这个蛋有什么特别的？"特殊花纹（非教学蛋，是惊喜蛋）
偷取并逃跑"我能拿到！"悬念保留——蛋的价值在孵化时揭晓，不提前告知
巢穴主人登场"完了，这下要打了"精英敌人 → 意图展示利用得好，短回合获胜；利用差战斗回合拉长，但都是胜利
精英战后"我能赢是因为我理解了它的意图"成长感 + 成就感 → 掉落一个新情报（暗示新内容）

4. 前15分钟的"甜蜜点"设计

第一场战斗（被气味吸引的生物）
  • 难度：极低，2-3回合内可击败
  • 教学目标：让玩家学会"点击攻击"和"释放技能"
  • 叙事包装：另一个生物也在追踪同一气味，双方因竞争气味源而发生冲突
  • 意图设计：生物有一个意图清晰的技能（如"爪击，造成15点物理伤害"），让玩家体验"我能读懂意图"的感觉
精英战斗（巢穴主人）
  • 难度：普通，4-8回合结束战斗
  • 教学目标：验证玩家是否理解"读取意图→选择对策"的战斗逻辑
  • 结果设计：意图展示利用得好，短回合获胜；利用差战斗回合拉长，但都是胜利
  • 战利品：获得一个新区域的位置情报（地图上解锁一个问号标记）

5. 第一只宠物的情感锚点设计

→
关键设计：玩家获得的第一只宠物不是教学赠送，而是通过偷蛋获得的惊喜。

蛋的出现时机：追踪气味的途中，不是主动送上门
蛋的稀有度暗示：纹路特别（但不明说价值）
蛋的揭晓时机：孵化时才看到宠物全貌
情感连接：玩家"偷"来的蛋比"送"来的蛋更有情感价值

6. 防止前期流失的关键

1
叙事驱动 > 机制教学
玩家不是在"学战斗"，而是在"追踪气味"的过程中自然学会战斗。
2
惊喜必须出现
宠物蛋的发现是情感锚点，必须在前15分钟内触发。
3
精英战斗必须可赢
难度要落在甜蜜点，需要理解意图但不至于劝退。
4
悬念保留到最后
蛋的价值不提前揭晓，让玩家带着"里面是什么？"的好奇心继续。

7. 常见失败模式（来自竞品教训）

失败模式后果
强制引导打断叙事玩家"被动学习"而非"主动探索"
教学道具直接赠送宠物没有情感价值，玩家记不住
精英战斗难度失衡失败即劝退，没有2-3次尝试的空间
奖励提前揭晓"偷蛋成功=获得稀有宠物"破坏悬念
核心原则

塞尔达设计的核心是"相信玩家的好奇心"——不要用强制引导消耗耐心，用精心设计的"甜蜜点"让玩家自己顿悟。"战斗即推理"的前期设计天然适合这个哲学：意图可视化本身就是一个值得"发现"的机制，而非需要"学习"的系统。
