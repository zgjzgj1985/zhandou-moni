## 一、设计背景

### 1.1 流派定位

草属性在循迹之境的8种属性体系中，定位为**光环流/增益流**流派。

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **草** | 光环流 / 增益流 | 厚积薄发的自然之力，以时间换力量 | 后期爆发 | ★★★☆☆ |

### 1.2 设计理念（修订版）

> *一句话定位：光能汇聚，芬芳绽放，万物生长*

**核心机制**（新增/修订）：
- **光能汇聚**：使用其他草系技能后，核心技能威力永久提升（无上限）
- **芬芳环境**：召唤特殊环境，提供额外增益效果
- **养分循环**：每回合回复能量，实现持续输出
- **层数叠加**：每回合提供持续收益，层数越高越强

**节奏特征**：光能汇聚 → 芬芳绽放 → 万物生长 → 收割阶段

**战斗哲学**：
- 用时间换取力量，拖得越久越强
- 光能不断积累，伤害无限成长
- 寄生战术消耗对手，持续回复保持战线

### 1.3 新增核心概念

#### 1.3.1 芬芳环境

芬芳环境是草系独有的战场状态，效果包括：
- 己方草系技能伤害 **+25%**
- 己方草系单位每回合额外回复最大HP的 **5%**

#### 1.3.2 光能汇聚

核心机制：使用其他草系技能后，下次使用的草系输出技能威力永久提升。

| 光能层数 | 威力加成 |
|----------|----------|
| 1层 | +60威力（首次触发） |
| 2层 | +120威力（累计） |
| 3层 | +180威力（累计） |
| N层 | +60×N威力 |

**注意**：光能汇聚只增强**草系输出技能**，不增强辅助/防御技能。

## 二、完整技能设计（12种）

### 2.1 攻击倾向技能（5种）

#### 【攻击倾向1】纤维化

| 属性 | 值 |
|------|-----|
| **技能ID** | `fiber_weave` |
| **技能名称** | 纤维化 |
| **能量消耗** | 2能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 40（低威力） |
| **伤害类型** | 特殊伤害（草属性） |

**效果描述**：攻击单体目标，造成40威力草属性伤害，获得1层「光能汇聚」

**核心机制**：
- 造成伤害后，自身获得1层「光能汇聚」
- 「光能汇聚」：下次使用的草系输出技能威力永久+60
- 是光环流的核心启动技能，用于积累光能

**设计理由**：
- 40威力对应2能量，看似亏模
- 但配合「光能汇聚」机制，后续技能获得巨额加成
- 是草系伤害不断膨胀的核心引擎

| **目标类型** | 自身 |
| **持续回合** | 3回合 |

## 五、战斗流程配合

### 5.1 推荐起手策略

| 回合 | 建议技能 | 理由 |
|------|----------|------|
| 第1回合 | 光能聚集（1能量） | 积累光能 |
| 第2回合 | 纤维化（2能量）+ 光能聚集（1能量） | 叠光能层数 |
| 第3回合 | 纤维化（2能量）+ 光能聚集（1能量） | 继续叠层 |
| 第4回合 | 韶光（8能量）/ 芬芳绽放（8能量） | 建立/刷新环境 |
| 第5回合 | 光能爆轰（2能量）- 消耗3层 = 280威力 | 收割阶段 |

### 5.2 光能汇聚连招

```
标准爆发连招：
第1回合：光能聚集（1能量）- 光能层数1
第2回合：纤维化（2能量）- 40伤害 + 光能层数2
第3回合：光能聚集（1能量）- 光能层数2
第4回合：纤维化（2能量）- 40伤害 + 光能层数3
第5回合：光能爆轰（2能量）- 消耗3层 = 100 + 180 = 280威力！
```

### 5.3 消耗流连招

```
长期消耗连招：
第1回合：寄生之种（3能量）
第2回合：扎根之躯（2能量）
第3回合：养分汲取（3能量）- 治疗+充能
第4回合：继续寄生之种叠加
第5回合：对手已被消耗致死
```

## 七、核心机制详解

### 7.1 光能汇聚机制

**触发条件**：使用「纤维化」「光能聚集」后获得

**效果**：下次使用的草系**输出技能**威力+60（每层）

**清空条件**：
- 使用「光能爆轰」消耗所有层数
- 离开战斗后清空

**可叠加**：无上限（理论上）

### 7.2 芬芳环境机制

**召唤方式**：「芬芳绽放」「韶光」

**持续时间**：
- 芬芳绽放：4回合
- 韶光：4回合（刷新时重新计时）

**叠加效果**：不可叠加，后使用者刷新时间

**环境效果**：
1. 己方草系技能伤害 +25%
2. 己方草系单位每回合回复5%HP

### 7.3 养分循环机制

**核心效果**：每回合回复能量（通过特定技能/特性）

**在当前设计中的体现**：
- 「养分汲取」可为目标回复4能量
- 配合光能汇聚机制，可实现持续高频输出
