## 一、设计背景

### 1.1 流派定位

超能属性在循迹之境的8种属性体系中，定位为**奥秘流/心理博弈**流派。

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **超能** | 奥秘流 / 心理博弈 | 心灵异能的迷雾，让对手在猜谜中犯错 | 心理博弈 | ★★★★★ |

### 1.2 设计理念

> *一句话定位：心灵异能的迷雾，让对手在猜谜中犯错*

**核心机制**：
- **延迟触发**：效果在N回合后触发或满足条件时触发
- **信息操控**：干扰敌方对意图图标的解读
- **心理博弈**：利用信息不对称制造决策压力

**节奏特征**：前期布阵（陷阱期）→ 中期发酵 → 后期爆发

### 1.3 流派克制关系

| 攻击流派 | 克制 | 被克制 | 设计理由 |
|----------|------|--------|----------|
| **奥秘流（超能）** | 中速流 | 连击流 | 布阵被打断 |

### 2.1 攻击倾向技能（3种）

#### 【攻击倾向1】迷心刺

| 属性 | 值 |
|------|-----|
| **技能ID** | `mind_pierce` |
| **技能名称** | 迷心刺 |
| **能量消耗** | 2能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 55 |
| **伤害类型** | 特殊伤害（超能属性） |

**效果描述**：攻击单体目标，造成55威力超能伤害，附加「心灵创伤」

**附加效果**：
- 使目标陷入「心灵创伤」状态（持续1回合）
- 心灵创伤：攻击命中率50%击中自己

**设计理由**：
- 低威力（55）对应中等消耗（2能量）
- 软控制效果需要后续攻击配合
- 心灵创伤是混乱的变体，增加心理压力

**代码实现**：
```typescript
export const MIND_PIERCE: Skill = (() => {
  const definition: SkillDefinition = {

    type: 'action',
    energyCost: EnergyCost.MEDIUM,
    target: SkillTarget.SINGLE,

    effects: [{
      damage: {
        basePower: 55,
        damageType: DamageType.SPECIAL,
        element: ElementType.PSYCHIC
      }
    }],
    category: '超能奥秘流·攻击',
    tags: ['超能', '奥秘流', '攻击', '软控制']
```
