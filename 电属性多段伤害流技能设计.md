## 一、设计背景

### 1.1 流派定位

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **电** | 电磁脉冲流 / 蓄能爆发流 | 先手压制+脉冲爆发，用速度积累电荷，用爆发收割战场 | 蓄能→先手压制→脉冲收割 | ★★★★★ |

### 1.2 设计理念

> *一句话定位：雷霆蓄于身，一击破千军*

**核心机制**（重写版）：

- **电荷积累**：电系技能命中时积累「电荷」层数
- **脉冲爆发**：电荷达到阈值时触发「电磁脉冲」，造成额外范围伤害
- **先手压制**：电系单位天生高速，用先手优势积累电荷
- **麻痹控制**：部分技能附带麻痹效果，阻断对手节奏

**节奏特征**：先手命中→积累电荷 →脉冲收割 →持续压制

**战斗哲学**：

- 用速度抢得先手，每一击都在积累力量
- 电荷不满时不轻易触发脉冲，保持威力
- 脉冲爆发时一击定胜负

### 1.3 与旧版设计对比

| 维度 | 旧版 v1.0（已废弃） | 新版 v2.0 |
|------|---------------------|------------|
| **核心机制** | 特性简单触发（每次技能40威力） | 电荷积累→脉冲爆发的节奏感 |
| **特性设计** | 4个特性互相独立，无配合 | 围绕「电荷」系统设计，相互协同 |
| **技能数量** | 12个（删减后） | 8个（精简实用） |
| **策略深度** | 无脑触发，缺乏决择 | 蓄能vs爆发的节奏博弈 |
| **视觉效果** | 缺乏特色 | 脉冲、闪电链等明确视觉 |

## 二、核心机制详解

### 2.1 电荷系统（核心创新）

电荷是电属性独有的资源系统，贯穿整个流派设计

#### 2.1.1 电荷积累

| 触发条件 | 积累层数 | 说明 |
|----------|----------|------|
| 电系技能命中单体 | +1层 | 基础积累 |
| 电系技能命中多个目标 | +2层 | 范围加成 |
| 先手攻击命中 | +1层（额外） | 速度奖励 |
| 目标带有「静电」debuff | +1层（额外） | 协同奖励 |

#### 2.1.2 电荷上限与清除

- **上限**：5层（可设计为可升级属性）
- **清空时机**：
  1. 使用「电磁脉冲」技能时全部清空
  2. 离开战斗时清空
  3. 回合结束时不自动清空（区别于草属性的光能）

#### 2.1.3 电荷效果

| 层数 | 技能威力加成 | 特殊效果 |
|------|--------------|----------|
| 0层 | +0% | - |
| 1层 | +10% | - |
| 2层 | +20% | - |
| 3层 | +35% | - |
| 4层 | +50% | 「超载」：下一次攻击附带连锁 |
| 5层 | +70% | 「临界」：电磁脉冲伤害翻倍 |

### 2.2 电磁脉冲机制

电磁脉冲是电属性核心的爆发技能，消耗所有电荷造成伤害

#### 基础伤害计算

\\\

脉冲伤害 = 基础威力 × (1 + 电荷层数 × 0.15) × 临界倍率

\\\

#### 脉冲类型

| 类型 | 触发条件 | 伤害 | 范围 |
|------|----------|------|------|
| 单体脉冲 | 电荷≥1层 | 100% | 单体 |
| 连锁脉冲 | 电荷≥3层 | 80% | 主目标+1个相邻目标 |
| 雷暴脉冲 | 电荷=5层（临界） | 150% | 主目标+所有相邻目标 |

## 四、完整技能设计（8种）

### 4.1 攻击倾向技能（3种）

#### 【攻击倾向1】电光一闪

| 属性 | 值 |
|------|-----|
| **技能ID** | \zap_strike\ |
| **技能名称** | 电光一闪 |
| **能量消耗** | 1能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 45（物理） |
| **伤害类型** | 物理伤害 |
| **标签** | \priority\, \charge_gain\ |

**效果描述**：如闪电般快速的斩击，必定先手攻击，造成45威力物理伤害。若命中，积累1层电荷。

**核心机制**：

- 先手+1：必定比对手先行动
- 电荷积累：命中后+1层电荷
- 配合「静电亲和」特性可+2层电荷

**设计理由**：

- 1能量/45威力 + 先手+1，低消耗启动技能
- 高速单位使用此技能可以不断积累电荷
- 先手保证每次都能命中获取电荷

#### 【攻击倾向2】雷鸣击

| 属性 | 值 |
|------|-----|
| **技能ID** | \thunder_strike\ |
| **技能名称** | 雷鸣击 |
| **能量消耗** | 3能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 90（特殊） |
| **伤害类型** | 特殊伤害（电属性） |
| **标签** | \charge_gain\, \paralyze\ |

**效果描述**：释放强力电击，造成90威力特殊伤害，命中后积累1层电荷，30%概率使目标麻痹1回合。

**核心机制**：

- 电荷积累：命中后+1层电荷
- 麻痹控制：30%概率使目标麻痹1回合
- 高威力输出技能

**设计理由**：

- 3能量/90威力，标准高消耗技能
- 麻痹效果提供控场能力
- 与「电光一闪」形成能量梯度

#### 【攻击倾向3】电磁脉冲（核心技能）

| 属性 | 值 |
|------|-----|
| **技能ID** | \electromagnetic_pulse\ |
| **技能名称** | 电磁脉冲 |
| **能量消耗** | 4能量 |
| **目标类型** | 单体敌人 |
| **技能威力** | 100（物理） |
| **伤害类型** | 物理伤害 |
| **标签** | \pulse\, \charge_consume\, \critical_potential\ |

**效果描述**：释放电磁脉冲，造成100威力物理伤害。消耗所有电荷层数，每消耗1层电荷，攻击次数+1。若电荷达到5层（临界），伤害翻倍并波及相邻目标。

**核心机制**：

- 电荷消耗：清空所有电荷层数
- 威力加成：基于电荷层数计算
- 临界效果：5层时伤害×2 + 范围伤害
- 连锁效果：临界时额外攻击1个目标

**设计理由**：

- 4能量/100基础威力，需要蓄能才有价值
- 临界状态提供爆发上限
- 波技能消耗所有蓄能，一击定胜负

**伤害表**：

| 电荷层数 | 威力倍率 | 伤害（100基础） | 效果 |
|----------|----------|-----------------|------|
| 0层 | ×1.0 | 100 | 基础伤害 |
| 2层 | ×1.3 | 130 | 中等爆发 |
| 3层 | ×1.45 | 145 | 触发连锁 |
| 4层 | ×1.6 | 160 | 超载（连锁） |
| 5层 | ×2.0+ | 200+ | 临界（翻倍+范围） |

### 4.2 防御倾向技能（2种）

#### 【防御倾向1】蓄电护体

| 属性 | 值 |
|------|-----|
| **技能ID** | \static_charge\ |
| **技能名称** | 蓄电护体 |
| **能量消耗** | 2能量 |
| **目标类型** | 自身 |
| **技能类型** | 防御 |
| **标签** | \charge_gain\, \shield\ |

**效果描述**：积蓄电能，获得护盾（30点）+ 积累2层电荷。本回合受到攻击时，攻击者获得1层「静电」debuff。

**核心机制**：

- 护盾获取：获得30点临时护盾
- 电荷积累：+2层电荷
- 静电反伤：攻击者获得「静电」debuff

**设计理由**：

- 2能量换取护盾+蓄电，一石二鸟
- 「静电」debuff为后续技能提供协同
- 防御技能也能积累电荷

|------|-----|
| **技能ID** | \electromagnetic_deflect\ |
| **技能名称** | 电磁偏转 |
| **能量消耗** | 3能量 |
| **目标类型** | 自身 |
| **技能类型** | 防御 |
| **标签** | \dodge\, \counter\ |

- 3能量换取一次高概率闪避
- 反击伤害虽低，但可以积累电荷
- 提供防御选择

## 十、平衡性调整建议

### 10.1 初期（测试阶段）

- 电荷积累：略微降低（0.5层/次）
- 电磁脉冲基础威力：适当降低
- 临界倍率：谨慎调整

### 10.2 中期（根据数据）

- 根据胜率调整各技能能量消耗
- 根据场均回合数调整电荷上限
