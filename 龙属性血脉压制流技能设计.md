## 一、设计背景

### 1.1 流派定位

龙属性在循迹之境的8种属性体系中，定位为**成长爆发流**。

| 属性 | 核心流派 | 定位描述 | 胜利节奏 | 操作难度 |
|------|----------|----------|----------|----------|
| **龙** | 成长爆发流 | 血脉传承的远古霸者，积累越久越强 | 滚雪球 → 一击必杀 | ★★★★☆ |

### 1.2 设计理念

> *一句话定位：血脉传承的远古霸者，积累越久越强*

**核心机制**：
- **血脉积累**：每回合自动叠加龙之气息，层数越高技能越强
- **多段打击**：龙系特有的鳞片连击，快速叠加层数
- **龙属共鸣**：与龙属性队友产生血脉共振，获得额外加成
- **流星爆发**：积累到顶峰时的究极爆发，代价是暂时虚弱

**节奏特征**：
```
第1-3回合（血脉觉醒期）→ 血脉积累
        ↓
第4-6回合（龙属共鸣期）→ 队友联动，滚大雪球
        ↓
第7回合+（流星陨落期）→ 究极收割或龙之终焉
```

### 1.3 克制关系

| 克制 | 被克制 | 免疫 |
|------|--------|------|
| 龙、超能 | 龙自身、冰 | 无 |

### 1.4 与宝可梦龙系的对照

| 宝可梦技能 | 本游戏对应 | 设计理念 |
|------------|------------|----------|

| 流星群 | 流星陨落 | 超高威力+能力下降作为代价 |

## 二、完整技能设计（12种）

### 2.1 血脉成长技能（3种）

#### 2.1.1 血脉觉醒（核心成长技）

| 项目 | 内容 |
|------|------|
| **能量消耗** | 2 |
| **效果** | 获得2层「龙之气息」+ 本回合下次攻击必定暴击 |
| **强化效果** | 当场上存在龙属性队友时，额外获得1层「龙之气息」 |

**设计说明**：这是龙属性的核心成长技能，类似宝可梦的龙之舞，但增加了龙属共鸣的联动要求。

#### 2.1.2 龙鳞连击

| 项目 | 内容 |
|------|------|
| **能量消耗** | 2 |
| **效果** | 对目标造成3次20威力**物理伤害**，每次攻击后提升自身速度1级 |
| **消耗层数** | 消耗3层「龙之气息」，每层增加10威力 |

**设计说明**：参考宝可梦的鳞片噪音（Scale Shot），多段攻击可以快速叠加速度并消耗层数增强威力。

#### 2.1.3 龙属共鸣（被动）

| 项目 | 内容 |
|------|------|
| **类型** | 被动技能 |
| **效果** | 每当友方龙属性单位行动时，获得1层「龙之气息」；自身攻击时，消耗所有层数，每层增加15威力 |

**设计说明**：这是龙属性的核心共鸣机制，让龙队形成滚雪球效应。

### 2.2 高威力攻击技能（4种）

#### 2.2.1 龙之波动

| 项目 | 内容 |
|------|------|
| **能量消耗** | 2 |
| **威力** | 80 |
| **效果** | 无副作用，稳定高伤害（**特殊伤害**） |

**设计说明**：宝可梦的经典技能，稳定的高威力输出。

#### 2.2.2 龙之终焉

| 项目 | 内容 |
|------|------|
| **能量消耗** | 4 |
| **基础威力** | 100 |
| **效果** | 造成伤害后使目标陷入2回合「混乱」（**物理伤害**） |
| **消耗层数** | 每消耗1层「龙之气息」，威力+15，混乱回合+1（最多3回合） |
| **龙属共鸣加成** | 当层数≥5时，混乱效果被「龙属共鸣」抵消 |

**设计说明**：参考宝可梦的逆鳞（Outrage），高威力但有失控副作用。通过消耗层数和龙属共鸣来平衡风险。

#### 2.2.3 流星陨落

| 项目 | 内容 |
|------|------|
| **能量消耗** | 5 |
| **威力** | 150 |
| **效果** | 对单体造成超极高伤害，使用后自身攻击力下降2级、特攻下降2级（**特殊伤害**） |
| **消耗层数** | 每消耗2层「龙之气息」，威力+30 |

**设计说明**：参考宝可梦的流星群（Draco Meteor），究极爆发但代价巨大。必须在天命期内使用才有意义。

#### 2.2.4 龙之碾压

| 项目 | 内容 |
|------|------|
| **能量消耗** | 3 |
| **基础威力** | 60 |
| **效果** | 目标每损失10%生命值，威力增加20%（**物理伤害**） |
| **特殊效果** | 若目标HP低于30%，必定命中且必定暴击 |

**设计说明**：参考宝可梦的龙之俯冲，对低血量目标有斩杀效果。

### 2.3 AOE与控制技能（3种）

#### 2.3.1 龙息灼烧

| 项目 | 内容 |
|------|------|
| **能量消耗** | 3 |
| **威力** | 50（全体） |
| **效果** | 对所有敌方单位造成伤害，并附加2回合「灼烧」状态（每回合受到15%最大生命值的伤害）（**特殊伤害**） |
| **消耗层数** | 每消耗1层「龙之气息」，灼烧伤害增加5% |

**设计说明**：参考宝可梦的龙息（Dragon Breath），AOE+持续伤害。

#### 2.3.2 龙之驱逐

| 项目 | 内容 |
|------|------|
| **能量消耗** | 2 |
| **效果** | 强制将目标与后排非战斗单位交换位置；若目标无后排可交换，则造成80威力伤害并使其攻击-2级（**物理伤害**） |
| **消耗层数** | 每消耗2层「龙之气息」，伤害增加20 |

**设计说明**：参考宝可梦的龙尾（Dragon Tail），强制换人机制可以打乱敌方阵型。

#### 2.3.3 龙威震慑

| 项目 | 内容 |
|------|------|
| **能量消耗** | 2 |
| **效果** | 使所有敌方单位攻击下降1级、速度下降1级，持续2回合 |
| **龙属共鸣加成** | 若自身「龙之气息」层数≥4，效果范围扩大至全体，效果持续时间+1回合 |

**设计说明**：群体软控制技能，强化龙属共鸣的收益。

### 2.4 防御与共鸣技能（2种）

#### 2.4.1 龙鳞守护

| 项目 | 内容 |
|------|------|
| **能量消耗** | 2 |
| **效果** | 本回合受到的所有伤害降低75%，并获得基于「龙之气息」层数的护盾（每层15点） |
| **特殊效果** | 护盾存在期间，若受到攻击，反击30威力伤害 |

**设计说明**：龙属性的防御技能，通过护盾+反击实现攻守平衡。

#### 2.4.2 龙属共鸣·极（终极技）

| 项目 | 内容 |
|------|------|
| **能量消耗** | 4 |
| **触发条件** | 场上至少存在2只龙属性友方单位 |
| **效果** | 消耗所有「龙之气息」层数，每层提供：15威力、10护盾、10%伤害减免 |
| **终结效果** | 若层数≥10，额外使所有敌方单位攻击力下降2级，持续3回合 |

**设计说明**：龙属性的终极联动技，层数越高效果越强。需要龙队配合。

## 四、战斗流程配合

### 4.1 典型回合规划

| 回合 | 推荐技能 | 目标层数 | 理由 |
|------|----------|----------|------|
| 第1回合 | 血脉觉醒 × 2 | 4层 | 快速建立层数优势 |
| 第2回合 | 龙属共鸣（被动） + 龙鳞连击 | 6层 | 叠层+消耗部分层数增强 |
| 第3回合 | 血脉觉醒 + 龙威震慑 | 8层 | 辅助队友+控制 |
| 第4回合 | 龙属共鸣·极（若双龙） | 10+层 | 触发终结效果 |
| 第5回合 | 龙之终焉 / 龙之碾压 | 消耗层数 | 斩杀低血目标 |
| 第6回合 | 流星陨落（若层数充足） | 爆发 | 究极收割 |

### 4.2 龙属共鸣阵容推荐

```
阵容A：纯龙队（最大化层数）
├── 龙属核心（血脉觉醒+龙属共鸣被动）
├── 龙属输出（龙之终焉+龙鳞连击）
└── 龙属控制（龙威震慑+龙之驱逐）

阵容B：龙+超能（控制联动）
├── 龙属核心
├── 超能辅助（精神干扰配合龙威）
└── 龙属爆发（流星陨落）

阵容C：龙+冰（克制关系）
├── 龙属核心
├── 冰属性控制（冰冻配合斩杀）
└── 龙属收割（龙之碾压）
```

## 六、平衡性调整记录

### v2.0 主要调整（2026-05-27）

1. **新增「龙之气息」层叠系统**：替代原有的固定加成，实现成长积累机制
2. **新增「龙属共鸣」系统**：与龙属性队友联动，增强收益

4. **新增「流星陨落」**：参考流星群，高风险高回报
5. **新增「龙鳞连击」**：多段攻击机制
6. **新增「龙之驱逐」**：强制换人机制
7. **新增「龙属共鸣·极」**：终极联动技
